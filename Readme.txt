1. Copy file RobloxMonitor-1.0.0-all.jar và start.bat vào 1 folder bất kỳ
2. Nhấn tổ hợp phím Windows + R để mở hộp thoại Run.
3. Gõ chính xác lệnh sau và nhấn Enter: shell:startup
4. Một thư mục tên là "Startup" sẽ mở ra. Bất cứ thứ gì nằm trong thư mục này sẽ được chạy khi bạn đăng nhập.
5. Quay lại thư mục C:\RobloxMonitor của bạn.
6. Nhấp chuột phải vào file start.bat -> Show more options -> Create shortcut (Tạo lối tắt).
7. Bạn sẽ có một file mới tên là start.bat - Shortcut.
8. Cắt (Cut) hoặc Kéo-thả (Drag-drop) file start.bat - Shortcut này vào thư mục Startup mà bạn đã mở ở Bước 4.
