Cài đặt Java 17 để sử dụng ứng dụng này
Hướng dẫn cài đặt tự động khởi động ứng dụng
1. Copy file AppMonitor-1.0-all.jar, start.bat, start_silent.vbs vào 1 folder bất kỳ
2. Nhấn tổ hợp phím Windows + R để mở hộp thoại Run.
3. Gõ chính xác lệnh sau và nhấn Enter: shell:startup
4. Một thư mục tên là "Startup" sẽ mở ra. Bất cứ thứ gì nằm trong thư mục này sẽ được chạy khi bạn đăng nhập.
5. Quay lại thư mục C:\AppMonitor của bạn (nơi bạn vừa copy 3 file trên).
6. Nhấp chuột phải vào file "start_silent.vbs" -> Show more options -> Create shortcut (Tạo lối tắt).
7. Bạn sẽ có một file mới tên là "start_silent.vbs - Shortcut".
8. Cắt (Cut) hoặc Kéo-thả (Drag-drop) file "start_silent.vbs - Shortcut" này vào thư mục Startup mà bạn đã mở ở Bước 4.
9. Khởi động lại máy
10. Sau khi ứng dụng khởi động thì mở ứng dụng và đổi mật khẩu từ "admin" thành mật khẩu của bạn
12. Chuột phải vào ứng dụng, chọn "Manage" -> Nhập mật khẩu mới
13. Ở tab Applications chọn "Add Running App", tìm đến chrome.exe (màn hình có thể hiển thị nhiều hơn 1 lựa chọn cho chrome.exe, hãy chọn 1 dòng bất kỳ) chọn "Select"
14. Ở tab WebSites chọn Add Website Manually, gõ vào keyword bạn muốn theo dõi xong chọn OK để kết thúc
15. Tắt cửa sổ Manage (Nút x ở góc trên)
16. Ứng dụng sẽ tự động hiện 1 cửa sổ ở góc phải trên khi phát hiện có ai đó truy cập vào các trang web hoặc ứng dụng đã cài đặt
17. Muốn ẩn (hide) cửa sổ theo dõi, chọn vào ứng dụng và chọn "Hide Overlay"
